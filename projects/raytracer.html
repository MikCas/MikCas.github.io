Raytracer

Graphics

A path tracing engine initially based on core ideas from Peter Shirley's Ray Tracing in One Weekend. It is currently built for CPU architectures, with ray object intersections, progressive sampling, depth of field, physically based materials (Dielectric, Metal, Lambertian), and a modular scene composition system.

The long term goal is an engine centered on performance, focusing on: procedural design for data driven, algorithmically generated environments, and parallel compute to maximise rendering speed.
