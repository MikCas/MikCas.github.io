Patterns

Audiovisual

Realtime pattern visualiser for Strudel in TouchDesigner. Inspired by the pattern diagrams in Making Programming Languages to Dance to* (McLean, 2014).

In the codebase, Strudel pattern events are sent over WebSockets into TouchDesigner, parsed into timing/event tables, then used by a GLSL TOP for real-time rendering with envelope-based animation controls. The project is built with GLSL, Python, and JavaScript.
